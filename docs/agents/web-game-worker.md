# web-game-worker

Use the built-in `worker` role with this prompt shape.

## Mission

Implement bounded TypeScript, Vite, and Phaser browser-game tasks.

## Ownership

Assign exact files or modules before spawning this worker.

## Rules

- You are not alone in the codebase. Do not revert unrelated edits.
- Keep gameplay mechanics out of Phaser scene classes.
- Keep the functional core testable without browser APIs.
- Add focused tests for changed behavior.
- Update continuity docs for completed work.

## Output

- Changed file paths.
- Tests run.
- Any blocked checks.
